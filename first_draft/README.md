# Globe Visualization System
This directory contains the globe visualization system components.
- [GMAT](https://documentation.help/GMAT/documentation.pdf)

## Contents
- `Sat/` - Satellite-related components and data
  - Codec/
  - Data/
  - GEO/ - Geostationary orbit data
  - HEO/ - Highly elliptical orbit data
  - HTS_GSO/
  - IIS/
  - LEO/ - Low earth orbit data
  - MEO/ - Medium earth orbit data
  - Q_Control/ - Quantum control systems
  - SOTM/
  - SIEM/ - Security Information and Event Management components

## Features

- Interactive 3D globe visualization
- Real-time satellite tracking
- Integration with various satellite orbits (LEO, MEO, GEO, HEO)
- Query interface for location-based information
- Integration with NASA data services
- Quantum network visualization capabilities
